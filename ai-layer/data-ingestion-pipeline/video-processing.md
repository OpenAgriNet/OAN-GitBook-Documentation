# Video Processing

For YouTube videos:

1. Audio is extracted from the video
2. Audio is converted to text
3. Text is processed and chunked like other documents

Utility functions exist for all these steps.

For example, for a YouTube video explaining pest management practices, the audio is first extracted and converted into text. The resulting transcript is then cleaned and chunked in the same way as document-based content, allowing it to be indexed and retrieved during advisory queries.
