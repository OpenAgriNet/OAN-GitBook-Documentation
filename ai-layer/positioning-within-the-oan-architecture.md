# Positioning Within the OAN Architecture

Within the overall OAN architecture, the AI module:

* Interfaces downstream with user-facing applications
* Interfaces upstream with content sources&#x20;
* Interfaces upstream  with knowledge repositories and external APIs

It acts as the intelligence layer that interprets different data sets integrated through a common protocol to generate a comprehensive response for the end user, while remaining fully aligned with OAN’s open, interoperable, and decentralised design principles.
