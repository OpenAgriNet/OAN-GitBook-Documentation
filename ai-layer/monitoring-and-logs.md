# Monitoring & Logs

The AI module includes monitoring and logging using tools such as Logfire. Logs capture:

* Query classification
* Tool calls
* Search terms
* Retrieved documents
* Final responses

This enables debugging, performance analysis, and iterative improvement of prompts and tools.

In addition to data quality checks, response outputs are periodically reviewed by domain experts to validate correctness, relevance, and safety. Feedback from these evaluations is used to refine prompts, tool logic, and moderation policies.
