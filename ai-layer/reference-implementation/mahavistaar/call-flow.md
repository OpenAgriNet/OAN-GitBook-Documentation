# Call Flow

At a high level, the interaction follows the sequence below:

1. Farmer calls MahaVISTAAR helpline
2. Voice query is captured and converted to text
3. Query is processed by AI agent and routed to appropriate tools
4. Tool responses are used to generate grounded responses
5. Response is converted to speech and delivered to the farmer
6. Session is maintained until user ends call
