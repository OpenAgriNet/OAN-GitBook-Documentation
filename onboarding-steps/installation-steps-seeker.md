| <p><em>Step 3:</em><br>Execute</p>                                     | <p>Run the Installation Script :<br>./beckn-onix.sh</p>                                                                                                                                                                          |
| <p><em>Step 4:</em><br><em>Selec</em> Participant Type(BAP/BPP)</p>    | Specify you want to join an existing network and install the BAP.                                                                                                                                                                |

| <p>Step 7: <br>Registry URL</p>                                        | Enter the address of the subscription endpoint of the registry of the network you want to join  ie.[ ](https://oan-registry.tekdinext.com/)[https://oan-registry.tekdinext.com/](https://oan-registry.tekdinext.com/)subscribers |
