# Pre-requisites

&#x20;_<mark style="color:blue;">( Please note many of the below items are Devops activities and hence it rqeuires the involvement of a Devops person )</mark>_

| Operating System | OS supported and tested Beckn-ONIX installer are Ubuntu (20.04, 22.04 and 24.04), Debian (12), Amazon Linux 2, Windows with WSL support and MacOS.                                                                                                                                                                                                                                                                                                                                                                                                     |
| ---------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Virtual Servers  | Virtual servers (EC2 instances) configured on the cloud provider with administrator login (e.g. ssh access) to them                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| Docker           | Latest docker version                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| Domain           | <p>You need to have 2 fully qualified domains.<br>If you are setting up the BAP/seeker side protocol adapter<br></p><p>eg. A domain or subdomain where the client endpoint of BAP/SEEKER will be accessible (e.g <a href="https://agriseeker-bap-client.mydomain_name.com">https://agriseeker-bap-client.mydomain_name.com</a>)</p><p><br>A domain or subdomain where the network endpoint of BAP/SEEKER will be accessible (e.g. <a href="https://agriseeker-bap-network.mydomain_name.com">https://agriseeker-bap-network.mydomain_name.com</a>)</p> |
|                  | <p><br>If you are setting up the BPP/provider side protocol adapter<br>eg. A domain or subdomain where the client endpoint of BPP/PROVIDER will be accessible (e.g <a href="https://agriseeker-bap-client.mydomain_name.com">https://agriseeker-bpp-client.mydomain_name.com</a>)</p><p><br>A domain or subdomain where the network endpoint of BPP/PROVIDER will be accessible (e.g. <a href="https://agriseeker-bap-network.mydomain_name.com">https://agriseeker-bpp-network.mydomain_name.com</a>)</p><p><br></p>                                  |
