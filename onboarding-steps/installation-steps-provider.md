It has to be an endpoint (it is called as webhook_url in the description below) which receives all the network requests (search, select etc) and send back the on_xxxx responses<br>Ref.Link: <br><br>1.  <a href="https://github.com/beckn/beckn-sandbox">https://github.com/beckn/beckn-sandbox</a>i<br>2. | --------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <p>Step 1 :<br>Clone the repo</p>                   | <p><strong>Clone the Beckn-ONIX repository :</strong><br>git clone https://github.com/beckn/beckn-onix.git</p>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <p>Step 2:<br>Go to the folder</p>                  | <p><strong>Navigate to the installation folder</strong> :<br>cd beckn-onix/install</p>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| <p>Step 3:<br>Execute</p>                           | <p><strong>Run the installation script:</strong><br>./beckn-onix.sh</p>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| <p>Step 4: <br>Select Participant Type(BAP/BPP)</p> | Specify you want to join an existing network and install the BPP.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| <p>Step 5:<br>Subscriber Id ie.URL w/o http</p>     | <p><strong>Enter the Subscriber ID for the BPP :</strong><br>eg. agriorg-bpp-network.mydomain_name.com<br><br><em><mark style="color:blue;">(Here please mention your domain name that will be registered in the OAN registry. The OAN gateway will broadcast any request from the network to this domain.</mark></em>   <br><em><mark style="color:blue;">Generally the subscriberId is  same as your domain URL without "https" )</mark></em></p>                                                                                                                                                                                                                                       |
| <p>Step 6: <br>Subscriber URL </p>                  | <p><strong>Enter the Subscriber URI for the BAP:</strong> </p><p><br><strong>eg.</strong> https://agriorg-bpp-network.mydomain_name.com<br></p>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <p>Step 7 :<br>Enter Registry Url</p>               | <p><strong>Enter the address of the subscription endpoint</strong> of the registry of the network you want to join</p><p><br> ie.<br><br><a href="https://oan-registry.tekdinext.com/login">https://oan-registry.tekdinext.com</a>/subscribers<br></p>                                                                                                                                                                                                                                                                                                                                                                                                                                    |
